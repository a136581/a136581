## 사용 기술
### Inverter/Converter
알맞은 사양의 회로를 개발할 수 있는 스킬을 가지고 있습니다.
- Simulator : PSIM, PSPICE, Matlab, Ansys, CATIA 등
- 회로설계 : Zuken/OrCAD 

### DSP Controller
하드웨어를 원격 제어할 수 있는 코딩 구현 스킬을 가지고 있습니다.
- C
- C++
- C#
- Python

### Sensor
센서를 사용해 피드백 받는 간단한 시스템 구현을 할 수 있습니다.
- 기능안전(ISO26262) 이해자
